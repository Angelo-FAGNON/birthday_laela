# Joyeux Anniversaire Läella

## Description

Application web interactive dédiée à la célébration d'un anniversaire. Le projet propose une expérience utilisateur immersive avec des éléments visuels sophistiqués, une gestion audio et une interface réactive.

## Caractéristiques

- **Design responsif** : Interface adaptée à tous les appareils (desktop, tablette, mobile)
- **Système audio intégré** : Gestion de pistes musicales avec palier d'activation/désactivation
- **Galerie visuelle** : Lightbox pour l'affichage des images en plein écran
- **Animations fluides** : Transitions et animations CSS optimisées
- **Thème personnalisé** : Palette de couleurs coordonnée (vert, or, blanc)

## Structure du projet

```
Birthday_laella/
├── index.html              # Page principale
├── assests/
│   ├── audio/             # Fichiers audio (mp3)
│   ├── css/
│   │   └── style.css      # Styles personnalisés
│   ├── img/               # Ressources graphiques
│   └── js/
│       └── scripts.js     # Logique applicative
└── README.md              # Documentation
```

## Technologies utilisées

- **HTML5** : Balisage sémantique
- **CSS3** : Styles responsifs avec variables CSS personnalisées
- **JavaScript (Vanilla)** : Manipulations du DOM et événements
- **Bootstrap 5** : Framework CSS pour la mise en page
- **Font Awesome** : Bibliothèque d'icônes

## Installation

1. Cloner ou télécharger le projet
2. Ouvrir `index.html` dans un navigateur web
3. Aucune dépendance serveur requise

## Utilisation

- **Musique** : Cliquer sur le bouton de musique pour activer/désactiver la lecture audio
- **Galerie** : Cliquer sur les images pour les afficher en plein écran via la lightbox
- **Navigation** : Utiliser les sections de la page pour explorer le contenu
